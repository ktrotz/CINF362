Challenge 1

Create two functions for the content below. When the "Summon Modal" button is clicked, one function should be triggered to show the modal content below (div with class of "modal hide"). Once the modal appears, the user should be able to click "Close Modal" to trigger another function that will close the modal/make it disappear. This challenge can be done in 9 lines of code or less.

Tips

  • You'll need two onclick functions (one for closeModal and one for summonModal).
  • Use the .classList.remove and .classList.add methods to add and remove classes
  • The class you will add and remove for each button is called "hide." When the summon modal button is clicked, remove hide. When close modal is clicked, add the class back to hide the modal again.

Challenge 2

Create a function which will allow a user to change the background color of the div with an id of "box" below. When a user types a color into the input area below and clicks "Submit Color", the background color of the box should change to that color. This challenge can be done in 5 lines of code or less.

Tips

  • Assign the button to a variable
  • Set that button to a onclick function
  • AFTER the button is clicked, get the input value
  • Change the background color with the .style method

Challenge 3

In the form below, you can use the inputs to select a number of each item in an imaginary ordering system. Create a function which will tell users how much of each item they ordered in addition to the total of their order. The output should be displayed on the page inside of the paragrah tag with the id of "orderTotal". An example of the output is displayed below. This challenge can be done in 10 lines of code or less.

"You have ordered 1 burger(s), 1 hotdog(s), 1 order(s) of fries, and 1 drink(s). Your total is $10.25."

Tips

  • Assign the button and output area (p tag) to variables
  • Get the quanity value AFTER they click submit
  • Use the "*" operator to perform math.
  • Output the whole order total as one concatenated string

Challenge 4

Create a function that will take in user input from the input box below. The user should enter a comma separated list of their hobbies. After they click submit, each hobby should appear on the page in a list. The list items should go directly inside of the ul tag with an id of "hobbiesContainer" below. If I enter "reading, gaming, traveling, and surfing the web" as my input and click "Submit Hobbies", a list of 4 items should appear. This challenge can be done in 9 lines of code or less.

Tips

  • Assign a variable to the button and unordered list (submitHobbies and hobbiesContainer).
  • Get the value of the hobby input AFTER submit is clicked.
  • Use the split method to split up their input by commas. This will create an array for you.
  • Use a for or while loop to go through the array and output each hobby. There is an example of looping through an array in the class notes.
  • You will need innerHTML to output to the page (using += so new content doesn't overwrite old content).
