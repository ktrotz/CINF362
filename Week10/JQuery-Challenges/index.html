JQuery Challenges

For this week, you have a few challenges to complete. Please don't add or remove HTML/CSS code as it will not be necessary for finishing these tasks. You should only be adding JQuery code to the script.js file included in the JQuery-Challenges.zip folder (JQuery reference is already included).Make sure you read everything carefully so you don't miss any requirements.

Challenge 1

Add code so that when the paragraph below is clicked, it disappears. You can achieve this effect with the hide() method.

Click to hide me.

Challenge 2

Add code so that when text is typed into the input below, that same text also appears inside of the empty paragraph below ("outputArea"). You'll need to use the keyup, html, and val methods in order to do this one.

Challenge 3

Add code so that when the button below is clicked, the width and height of the box is increased (200px is fine for both dimensions). You'll need the css method for this challenge.

Challenge 4

Add code so that when the button below is clicked, the image will fade in or fade out. You can accomplish this with the fade in/fade out or fade toggle methods.


Challenge 5

Add code so that whenever the button below is clicked, its text changes to tell the user how many times it has been clicked. After clicking it 5 times, the button should say "I've been clicked 5 times." You can use the html method to change the text of the button. You'll need to create a counter to keep track of how many times it has been clicked.
